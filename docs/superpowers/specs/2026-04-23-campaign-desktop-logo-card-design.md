# 캠페인 데스크탑 메인 카드 로고 배경 처리 디자인 문서

## 목표
캠페인 선택 화면의 데스크탑 메인 카드에 테마 로고 이미지를 배경 레이어로 배치해 카드의 개성을 강화한다.  
동시에 로고가 `title`, `description`, `button`을 가리지 않도록 어두운 오버레이와 하단 그라데이션, 후방 글로우 처리로 정보 계층을 분리한다.

## 확정 범위
- `CampaignScene`의 데스크탑 메인 `hero card`에만 적용한다.
- 모바일 메인 카드에는 적용하지 않는다.
- 좌우의 소형 프리뷰 카드에는 적용하지 않는다.
- 기존 카드의 텍스트, 버튼, 상태 칩 구성은 유지한다.
- 기존 테마별 로고 자산을 재사용한다.

## 현재 문제
- 현재 데스크탑 메인 카드는 단색 패널과 장식선 위주라 테마별 시각 차별점이 약하다.
- 로고는 별도 배경 이미지처럼 쓰이지 않고, 카드 정체성을 강하게 전달하지 못한다.
- 카드에 배경 이미지를 단순히 올리면 제목, 설명, 버튼과 충돌해 가독성이 떨어질 수 있다.

## 디자인 원칙
- 로고는 카드의 주인공이 아니라 콘텐츠를 받쳐주는 배경 레이어여야 한다.
- 텍스트와 버튼은 항상 로고보다 앞에 읽혀야 한다.
- 요청한 HTML 예시처럼 `img -> dark overlay -> bottom gradient -> foreground content` 순서를 Phaser 레이어로 재현한다.
- 데스크탑 메인 카드만 강화하고, 나머지 카드의 역할과 밀도는 그대로 유지한다.

## 레이어 구조
데스크탑 메인 카드 내부 구조는 아래 순서를 따른다.

1. 카드 패널 프레임과 외곽선
2. 로고 이미지 레이어
3. 어두운 배경 오버레이
4. 하단에서 위로 올라오는 그라데이션/마스크 성격의 쉐이프
5. 텍스트, 상태 칩, 버튼 등 전면 콘텐츠

이 구조는 사용자가 제시한 HTML의 `absolute inset-0` 배경 컨테이너와 동일한 시각적 의도를 Phaser 오브젝트 계층으로 옮긴 것이다.

## 로고 배치
- 로고 이미지는 카드 내부를 넉넉하게 덮는 크기로 배치한다.
- 중심점은 카드의 우측-중앙 또는 중앙-우측 쪽에 둔다.
- 제목과 설명이 자리 잡는 좌측 본문 영역과 직접 겹치지 않도록 한다.
- 로고는 `object-cover`에 해당하는 느낌으로 충분히 크게 보여 주되, 정보층과 경쟁하지 않도록 낮은 알파값을 사용한다.

## 후방 처리 방식
- 로고 위에 어두운 반투명 오버레이를 얹어 전체 명도를 낮춘다.
- 카드 하단에는 본문과 버튼이 놓인 영역을 보호하는 그라데이션성 쉐이프를 추가한다.
- 로고 뒤에는 테마 색상 기반의 부드러운 글로우를 남겨 너무 평평하게 죽지 않게 한다.
- 필요하면 로고는 현재 Phaser 범위에서 실제 블러 대신 저알파 + 글로우 + 오버레이 조합으로 후방 블러와 유사한 느낌을 만든다.

## 레이아웃 영향
- 데스크탑 메인 카드의 기존 텍스트 블록 배치와 버튼 위치는 유지한다.
- 다만 로고가 오른쪽 뒤에 들어오므로, 데스크탑 액션 레인과 텍스트 래핑 폭이 지금처럼 유지되는지를 함께 확인한다.
- 모바일과 컴팩트 데스크탑 분기는 기존 조건을 유지한다.

## 구현 방향
- 변경 파일은 우선 `src/phaser/scenes/CampaignScene.js`로 한정한다.
- 기존 `createThemeHeroCard` 내부에서 데스크탑 전용 배경 로고 레이어를 추가한다.
- 현재 사용 중인 테마 시길 이미지를 그대로 활용하되, 배경용 배치 값과 알파값을 별도로 조정한다.
- 그라데이션은 Phaser `Graphics` 기반의 다중 쉐이프 또는 지원 가능한 그라데이션 API로 구성한다.
- 카드의 전면 콘텐츠는 마지막에 추가해 항상 최상단에 오도록 유지한다.

## 오류 및 회귀 방지
- 잠금 상태와 클리어 상태 텍스트가 배경에 묻히지 않아야 한다.
- 버튼 라벨이 배경 로고와 겹쳐도 충분한 대비를 유지해야 한다.
- 카드 클릭과 버튼 클릭 인터랙션은 기존과 동일하게 동작해야 한다.
- 소형 프리뷰 카드와 모바일 카드의 시각 구조는 바뀌지 않아야 한다.

## 검증 기준
- 데스크탑 캠페인 메인 카드에 테마 로고가 배경 이미지처럼 보인다.
- 로고는 카드 뒤에 물러나 있고 제목, 설명, 버튼보다 튀지 않는다.
- 카드 하단의 제목/설명/버튼 가독성이 유지된다.
- 모바일 카드와 좌우 프리뷰 카드는 이전과 동일하게 유지된다.
- 기존 테마 전환, 브리핑 진입, 잠금 상태 표시는 정상 동작한다.
